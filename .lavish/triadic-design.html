Triadic 三色组配色方案

色轮上 120° 等间隔的三种颜色，高对比且平衡，适合活力风格的设计。

🌈 什么是 Triadic？

Triadic（三色组）是色轮上均匀间隔 120° 的三种颜色。相比 complementary 的强对比，triadic 在保持视觉张力的同时更均衡。

等边三角形连接色轮上的三个点

💡 API 设计

新增函数

// 生成 triadic 三色组
export function generateTriadicPalette(hex) {
  return {
    primary:   '#c06b5e',  // 基色 (0°)
    secondary: '#5ec06b',  // +120°
    tertiary:  '#6b5ec0',  // +240°
    light:     lighten(primary, 0.3),
    dark:      darken(primary, 0.3),
  };
}

复用已有函数

内部复用 rgbToHsl / hslToRgb / rotateHue（如已存在），否则新建这些辅助函数。
